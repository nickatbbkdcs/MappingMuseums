Database Browse, Search and Visualisation

This page will describe the main features and capabilities of the database Browse, Search and Visualisation facilities.

It will include a short Video showing how to use these facilities.

It will also include information about all of the hierarchies under which the museums data is organised, and a Glossary of all the terms used in the hierarchies.

Technical details: This Web Application has been developed using a three-tier architecture. The database server, which hosts the Mapping Museums Knowledge Base, is a Virtuoso server. The web client application is built using Leaflet, JQuery, Bokeh, Bootstrap and various widget javascript packages. The back-end is developed in Python/Flask/Jinja and deployed to Apache/WSGI; communication with the database server is through a SPARQL endpoint.

Further information is available in this pdf presentation from page 24 onwards.
